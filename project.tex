\normalpage{The Fox and the Crab}{by Alex Nguyen, Raymond Zerulla, Jiajun Liang, and Alexander Shtov}
\normalpage{cs102 Project 1: Specification}{
Our group project's goal is to make a program that will animate the 
Aesop's Fable, "The Fox and the Crab". This will include the animation made with vector graphics
and a textbox underneath this animation narrating the story. 
}
\normalpage{cs102 Project 1: Analysis}{
\qi{Start with a credits title page before the main story.}
\qi{The crab, fox, and sand mites are done with vector graphics.}
\qi{The background changes according to the scenes.}
\qi{The animation of the crab and fox is done with translation, rotation, and scaling.}
\qi{The caption is added underneath the animation.}
\qi{Sand mites jitter to simulate movement.}
\qi{Use PNG images for the backgrounds, fox face, and blood.}
\qi{Load all images at the start of the program to prevent lag.}
}
\normalpage{cs102 Project 1: Design - Assigned Parts}{
\qi{Main and draw\_cb: Alexander}
\qii{main - set up animation timer and FLTK window}
\qii{draw\_cb - animates everything according to the animation timer}
\qi{Background: Alex}
\qii{draw\_image - loads all images the first time it is called, afterwards just displays the image selected}
\qi{Crab: drawn using vector graphics - Jiajun}
\qii{draw\_crab - draws the body circle and calls functions to draw other parts}
\qii{draw\_crab\_legs}
\qii{draw\_crab\_pincer}
\qii{draw\_crab\_face - happy face}
\qii{draw\_crab\_exciteface - scared face}
\qii{draw\_crab\_sadface}
\qi{Sand Mites: draw using vector graphics - Alex}
\qii{draw\_sand\_mites - alternates position by 2 pixels every time it is called, simulating movement}
}
\normalpage{cs102 Project 1: Design - Assigned Parts cont.}{
\qi{Textbox: Sentences describing scenes in a window - Alexander}
\qii{draw\_caption - calls draw\_caption\_background and draw\_caption\_text}
\qii{draw\_caption\_background - draws a black box on the bottom of the screen to frame the caption}
\qii{draw\_caption\_text - draws the text for the scene on 1 or 2 lines}
\qi{Fox: Vector graphics - Ray}
\qii{draw\_fox - calls draw\_fox\_body and one of the two draw\_fox\_legs alternating every 2 times it is called}
\qii{draw\_fox\_body}
\qii{draw\_fox\_legs - 2 functions for 2 different positions of the legs} 
\qi{Zoom: Alexander}
\qii{Zoom scene: zooms into scene of fox eating crab}
\qii{Draws a fox head that eats the crab from a PNG file.}
}

\normalpage{cs102 Project : Problems}{
\qi{Loading PNG images from hard drive every frame caused lag}
\qii{Fixed by loading images only once when the program starts}
\qi{Fox, crab, and sand mites were not drawn at the origin originally}
\qii{Crab and sand mites were modified to be drawn at the origin for easy translation}
\qii{Did not have time to fix fox; worked around it}
\qi{Repeated problem with vector graphics for some reason having ``int a2 = PI * 2''}
\qii{PI is not an integer}
\qii{Caused graphical glitches (lines drawn to other parts of the scene)}
\qii{Fixed by changing all occurences to ``double a2 = PI * 2''}
}
